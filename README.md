# 🏠 Online Room and Home Booking System

## 📌 Overview
The Online Room and Home Booking System is a web-based application that allows users to search and book rooms and homes online. It provides a user-friendly and efficient platform for managing bookings digitally.

## 🎯 Objective
- Develop an online booking system  /// shreya contribution
- Provide user registration and login  
- Enable room search and booking  
- Implement billing and invoice generation  
- Provide feedback and rating system  
- Create admin panel for room management  
- Manage room availability (inventory system)  

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  

## ✨ Features
- 🔐 User Authentication (Login/Signup)  
- 🔍 Search available rooms  
- 🏠 Book rooms and homes  
- 🧾 Generate bills/invoices  
- ⭐ Feedback and rating system  
- 🛠️ Admin panel for room management  

## ⚙️ How It Works
Users register and log in → search rooms → book rooms → generate bills → give feedback.  
Admin manages rooms and bookings.

## 📂 Installation & Setup
1. Clone the repository  
2. Run `npm install`  
3. Setup `.env` file with MongoDB connection  
4. Run `npm start`  
5. Open browser at `http://localhost:3000`

## 🏗️ System Architecture
- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  

Frontend → Backend (API calls) → Database (MongoDB)
